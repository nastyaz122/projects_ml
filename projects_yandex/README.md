# Проекты Яндекс.Практикума

Здесь собраны проекты, выполненные в ходе обучения на курсе Яндекс.Практикума.

| Название проекта | Описание | Используемые библиотеки |
| :---------------------- | :---------------------- | :---------------------- |
| [Яндекс Музыка](./yandex_music.ipynb) | Сравнение предпочтений пользователей Яндекс.Музыки из Москвы и Санкт-Петербурга в зависимости от времени суток и дня недели | *pandas* |
| [Надежность заемщиков. Часть 1](./borrowers_1.ipynb) | Выявление зависимость для определения надежных заемщиков для выдачи кредита. Первая часть проекта. | *pandas* |
| [Надежность заемщиков. Часть 2](./borrowers_2.ipynb) | Выявление зависимость для определения надежных заемщиков для выдачи кредита. Вторая часть проекта.  | *pandas*, *matplotlib*, *seaborn* |
| [Исследование объявлений о продаже квартир](./apartments_for_sale.ipynb) | Выявление параметров и зависимостей на стоимость объектов недвижимости | *pandas*, *matplotlib* |
| [Исследование данных о российском кинопрокате](./russian_film_distribition.ipynb) | Исследование рынка российского кинопроката с целью выявления текущих трендов, с особым вниманием к фильмам, получившим государственную поддержку | *pandas*, *matplotlib* |
| [Статистический анализ сервиса аренды самокатов](./stat_data_analysis.ipynb) | Статистический анализ данных популярного сервиса аренды самокатов GoFast и проверка гипотез | *pandas*, *matplotlib*, *numpy*, *math*, *scipy* |
